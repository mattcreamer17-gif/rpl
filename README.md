# RPL International landing page

Single-purpose sales page built for Roberto Paredes Loewenberg and the RPL team, used at eMerge Americas 2026 (Booth 350). The page carries the "read on where AI fits in RPL's staffing practice" thesis, the 60 / 30 / 10 frame, the KPMG UK (Big Four) case study, the Interpretable Context Methodology paper, the Ethics Engine one-liner with links, the NLP Logix routing sentence with mandatory credibility stats, and a single CTA to Matt Creamer's Calendly.

Intended URL path: /for/rpl-international

Built with RPL's navy / gold / Montserrat / Open Sans brand tokens (from `Brand Pack/brand-pack.json`). No Eduba branding at the top. Eduba is referenced only in the footer and in named case-study context. Static HTML + CSS, no JavaScript. Mobile-tested at 375px.
